# Dropdown choices API

`GET /api/dropdown?field=AD_Org_ID&type=143&id=7765`

> **field** – field name for which we shall get choices
> **type** – document type identifier
> **id** – document entity ID

```json
[
    { "112": "Limited" },
    { "113": "Incorporated" },
    { "115": "Limited" }
]
```
